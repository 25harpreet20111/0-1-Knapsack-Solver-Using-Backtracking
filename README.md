# 0/1 Knapsack Problem Solver

## Introduction
This project demonstrates the 0/1 Knapsack problem using the Backtracking algorithm with a graphical interface built using Tkinter.

## Objectives
- Understand Backtracking algorithm
- Visualize item selection
- Calculate maximum profit

## Technologies Used
- Python
- Tkinter

## Features
- GUI Interface
- Dynamic Input
- Best Profit Calculation

## Algorithm
Backtracking Algorithm

## How to Run
python main.py
